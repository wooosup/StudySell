## 프로젝트 개요
- 대학생들이 학기마다 새로 구입해야 하는 교재 비용을 절감할 수 있도록 중고 거래를 통해 경제적 부담을 줄이는 것입니다. 
- 책이 전문적이고, 대중적이지 않은 책이기 때문에 판매하는 사람들이 많이 없어 대학 내에서 직접 거래할 수 있는 플랫폼을 제공하여,
학생들이 안전하고 편리하게 거래할 수 있도록 합니다.
- 가장 큰 이유는 계속 쌓여져 가는 제 전공 책들을 팔고 싶었습니다.
